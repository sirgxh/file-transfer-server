# 文件传输服务器-GXH

一款轻量级的局域网文件传输工具，基于 Python + Tkinter 开发，一键启动 HTTP 服务，让同一局域网内的用户通过浏览器或扫码即可上传文件到你的电脑。

## ✨ 功能特点

- 🚀 **一键启动** - 图形化界面，操作简单，无需复杂配置
- 📱 **二维码访问** - 生成二维码，手机扫码直接上传
- 🔐 **密码验证** - 支持 HTTP Basic Auth，可选开启
- 🔗 **随机URL路径** - 每次启动自动生成随机路径，安全隐蔽
- 📁 **自定义目录** - 可设置文件存放目录，默认「下载」目录
- 📊 **实时列表** - 显示本次接收的文件列表（文件名、类型、大小、时间、来源IP）
- 🖱️ **右键操作** - 右键打开文件或打开所在目录
- 📝 **传输日志** - 关闭时自动保存传输记录到日志文件
- 🔄 **批量上传** - 客户端支持拖拽和批量上传

## 🚀 快速开始

```bash
# 安装依赖（二维码功能需要）
pip install qrcode[pil]

# 运行
python httpserver.py
```

## ⚙️ 配置说明

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 监听地址 | `0.0.0.0` | 监听所有网卡，局域网用户都可访问 |
| 端口 | `80` | HTTP标准端口，1024以下需管理员权限 |
| 存放目录 | `~/Downloads` | 系统下载目录 |
| 访问路径 | 随机8位 | 随机URL路径，可手动修改或刷新 |
| 密码验证 | 关闭 | 可开启HTTP Basic Auth |

## 🔧 技术栈

- **语言**：Python 3
- **GUI**：Tkinter
- **Web服务器**：http.server（多线程）
- **认证**：HTTP Basic Auth
- **二维码**：qrcode + Pillow
